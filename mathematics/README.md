# Mathematics for Data Science

Welcome to the Mathematics section of our Data Science study group! This directory is dedicated to providing resources and materials for fundamental mathematical concepts needed in Data Science.

## Subtopics Structure

This directory is organized into various subdirectories, each dealing with a different aspect of Mathematics. Here's a brief overview of the subtopics:

- **Linear Algebra**
    - Matrix Theory
    - Vector Spaces
    - Eigenvalues and Eigenvectors
    - Determinant

- **Statistics**
    - Descriptive Statistics
    - Inferential Statistics
    - Probability Distributions
    - Hypothesis Testing

- **Calculus**
    - Differential Calculus
    - Integral Calculus
    - Multivariable Calculus
    - Differential Equations

- **Probability**
    - Combinatorics
    - Probability Theory
    - Conditional Probability and Bayes' Theorem
    - Random Variables

We encourage everyone to contribute to the directory by adding resources, asking questions, and providing answers. Happy learning!
